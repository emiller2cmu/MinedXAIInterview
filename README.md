# MinedXAIInterview
Emily Miller's Repo for the MinedXAI Interview 


## How to run: 

For the backend

### `cd frontend`

### `npm i`

### `npm start`

For the frontend

### `cd backend`

### `node server.js`

Runs the app in the development mode.\
Open [http://localhost:8081/](http://localhost:8081/) to view it in the browser.
